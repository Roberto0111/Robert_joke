# Daily Comic Style

每天產出一張新的 IG 六格漫畫，固定風格如下。

## 核心風格

- 類型：認真講幹話
- 格式：六格漫畫，2 欄 x 3 列
- 語言：繁體中文
- 畫風：彩色漫畫、乾式幽默、清楚分鏡、白色對話框
- 主角：同一位黑髮男性角色，黑色有領上衣，表情常常很認真
- 配角：如果有貓，一定是黑白賓士貓，負責冷眼吐槽或沉默

## 笑話節奏

最常用節奏：

1. 主角提出一個很認真的人生理論
2. 用荒謬邏輯拆解問題
3. 繼續把錯誤推論說得像真的
4. 貓或旁人開始懷疑
5. 主角講出主要 punchline
6. 沉默留白，讓荒謬感自己發酵

第 5 格通常是主笑點，第 6 格不要硬解釋。

## 好用句型

- 「不是 X，是 Y。」
- 「我不是在逃避，是在降低人生波動。」
- 「先不做，是保留成功的可能性。」
- 「這不是懶，是低功耗策略。」
- 「我沒有放棄，我只是提前跳過失敗流程。」

## 避免

- 不要重複已經發過的「不是偷懶，是提前抵達成果。」
- 不要一次生成多張。
- 不要一次生成多則笑話。
- 不要把 token、`.env`、access key 印出來。
- 不要把 `.env` commit 進 git。
