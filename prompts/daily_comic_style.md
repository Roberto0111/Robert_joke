# Daily Comic Style

每天產出一張新的 IG 六格漫畫，固定風格如下。

## 核心風格

- 類型：認真講幹話
- 格式：六格漫畫，2 欄 x 3 列
- 尺寸：固定 1080x1350，IG 4:5 portrait feed 安全比例
- 語言：繁體中文
- 畫風：彩色漫畫、IG 直式截圖友善、清楚分鏡、白色對話框、夜間辦公室或日常場景、表情冷靜但荒謬
- 主角：必須依照 `assets/main_character_reference.jpg` 的本人照片。特徵：東亞男性、圓臉、黑色旁分瀏海、眼神偏睏但冷靜、黑色有領上衣、灰色拉鍊/門襟。可以漫畫化，但不能變成 generic anime man。
- 配角：如果有貓，一定是黑白賓士貓，負責冷眼吐槽或沉默

## 喜劇口味

要更靠杯，不要溫馨。

主角的語氣像一個很認真的主管、顧問、會議主持人或人生管理師，但內容完全是幹話。笑點來自「把逃避、拖延、懶、社交失敗、工作混亂講成高級策略」。

核心感覺：

- 嚴肅地把小事制度化
- 把問題包裝成流程
- 把不想面對講成風險控管
- 把失敗說成提前完成驗證
- 把耍廢說成心理成本最佳化
- 貓用冷臉拆穿，或完全沉默讓畫面更尷尬

不要用太可愛、太正能量、太像勵志語錄的笑點。要有一點「靠北，這人怎麼可以講得這麼理直氣壯」的感覺。

## 笑話節奏

最常用節奏：

1. 主角提出一個很認真的人生理論
2. 用荒謬邏輯拆解問題
3. 繼續把錯誤推論說得像真的
4. 貓或旁人開始懷疑
5. 主角講出主要 punchline
6. 沉默留白，讓荒謬感自己發酵

第 5 格通常是主笑點，要像標語或結論一樣有力。第 6 格不要硬解釋，可以只放貓的冷眼、主角睡著、白板留下荒謬結論、或一個「……」。

每格文字要短，適合 IG 手機閱讀。不要塞滿台詞。每格最多 1 個主要對話框，必要時加一張小便條、白板或牌子補刀。

## IG 裁切安全

- 不要輸出 2:3、9:16 或過高長圖。
- 所有外框、人物臉、對話框、文字都要留在 4:5 內側安全區。
- 上下左右至少留一點呼吸空間，不要讓字或角色貼到圖片邊緣。
- 六格分鏡可以填滿畫面，但重要內容不能靠最外側 5% 區域。

## 常見場景

- 辦公桌、筆電、白板、便利貼、會議室
- 行事曆、待辦清單、登入畫面、鬧鐘、報表
- 主角正經坐著、指白板、拿手機、雙手交疊沉思
- 賓士貓坐在桌上，眼神厭世，偶爾舉牌吐槽

## 反轉規則

反轉不要靠無厘頭亂跳，而是用錯誤邏輯一路推到荒謬結論。

好反轉像這樣：

- 看起來在解決問題，其實是在替逃避找制度
- 看起來在提高效率，其實是在把事情合理地不做
- 看起來在自我管理，其實是在把懶惰升級成方法論
- 看起來在檢討人生，其實是在宣布人生不用檢討

## 好用句型

- 「不是 X，是 Y。」
- 「我不是在逃避，是在降低人生波動。」
- 「先不做，是保留成功的可能性。」
- 「這不是懶，是低功耗策略。」
- 「我沒有放棄，我只是提前跳過失敗流程。」
- 「不是拖延，是讓問題自然成熟。」
- 「不是沒做，是正在降低待辦事項的心理積秤。」
- 「問題如果太快解決，會沒有存在感。」
- 「我不是擺爛，我是在做壓力去中心化。」
- 「把事情放著，是給它一個自我修復的機會。」
- 「會議不是浪費時間，是讓問題有儀式感地活著。」

## 避免

- 不要重複已經發過的「不是偷懶，是提前抵達成果。」
- 不要一次生成多張。
- 不要一次生成多則笑話。
- 不要寫成普通冷笑話、諧音笑話、爸爸笑話。
- 不要讓主角大笑或做誇張搞笑表情；他要越正經越好笑。
- 不要讓貓太可愛或太活潑；賓士貓要冷、厭世、像看透一切。
- 不要用太多說明文字解釋為什麼好笑。
- 不要把 token、`.env`、access key 印出來。
- 不要把 `.env` commit 進 git。
